ProxyTester

Testa listas de proxies HTTP contra uma URL alvo, N vezes cada, com validação de conteúdo real.

A página alvo é baixada uma vez sem proxy, como referência. Cada resposta via proxy é comparada a ela; uma requisição só conta como sucesso se o conteúdo for pelo menos este % parecido (algum desvio é normal — anúncios e widgets dinâmicos variam entre requisições).
