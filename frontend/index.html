ProxyTester

Test HTTP proxy lists against a target URL, N times each, with real content validation.

Use HTTP Client for maximum speed. Use Headless Browser for sites requiring full JavaScript execution and browser rendering.

The target page is downloaded once without proxy as a reference. Each response via proxy is compared to it; a request only counts as success if the content is at least this % similar (some variation is normal — ads and dynamic widgets vary across requests).
